Simple Calendar

Este proyecto es el primero hecho en React.

A modo de practica decidi comenzar un mini proyecto que sea capaz de terminar y pulir.

Este calendario simple te proporciona un sistema de guardado para tus tareas diarias sin necesidad de registrarte en ningun lado, asegurando que tus tareas sean 100% privadas.

He empleado el paradigma first mobile al desarrollar esta app. Al estar pensada para ser una Progressive Web App fue especificamente adaptada para dichas resoluciones.
