Simple Calendar

Este proyecto es el primero hecho en React.

A modo de práctica decidí comenzar un mini proyecto que sea capaz determinar y pulir.

Este calendario simple te proporciona un sistema de guardado paratus tareas diarias sin necesidad de registrarte en ningun lado,asegurando que tus tareas sean 100% privadas.

He empleado el paradigma first mobile al desarrollar esta app. Alestar pensada para ser una Progressive Web App fueespecificamente adaptada para dichas resoluciones.
